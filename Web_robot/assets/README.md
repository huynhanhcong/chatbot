# Icon assets for Web_robot

Place your icon files in this folder with these names:

- `guide.png` for the Huong dan button
- `doctor.png` for the Bac si button
- `voice.png` for the Dam thoai button
- `play.png` for guide rows
- `manual.png` for manual medicine entry
- `qr.png` for QR scan
- `delete.png` for keypad delete
- `confirm.png` for keypad confirm
- `settings.png` for the doctor settings button

The app hides missing icon files automatically, so the interface still loads while icons are being prepared.
